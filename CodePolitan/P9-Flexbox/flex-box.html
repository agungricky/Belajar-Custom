P9 - Flax-Box

Disini kita akan belajar Tentang dispalay Flex-Box

* Display Flex *

  • disini bisa kita Lihat bahwasanya Flex bekerja dengan cara menyesuaikan
    lebar konter/wadahnya
  • Biasa di Gunakan Untuk Layar Responsiv
  • Cara Pembuktian : Coba Tambahka 1 Kotak lagi

* Float *

  • disini bisa kita Lihat bahwasanya Float bekerja dengan Cara Menggunakan Kotak dengan Ukuran Asli
    Jika Ruang dalam Kontainer tidak mencukupi maka, Kotak selanjutnya akan di taruh bawahnya
  • Biasa di gunakan Untuk Untuk Menu Bar
  • Cara Pembuktian : Coba Tambahka 1 Kotak lagi
