P1 Mengubah Konten Melalui Id

Pada gamabar di atas bisa kita lihat bahwasannya Text ditulis di dalam Java Script denagan cara mengganti id dari elemen tag "P"

Penjelasan : document.getElementById("konten").innerHTML = "Aku Sayang ika,, Sayangku";

  • document = Menulis Langsung pada Documen Html
  • getElementById("konten") = Mengambil id
  • innerHTML = berfungsi Untuk Menggantikan TEXT
  • = "Aku Sayang ika,, Sayangku"; = Kata / Text Yang akan di Tampilkan

Jenis-Jenis Penampil Text

Ada 3 cara Untuk menampilkan text yang kita Inpukan Atau Untuk Mengecek apakah komponen berfungsi dengan baik, berikut penampil textnya :

  • Console.log
  • Alert
  • document
