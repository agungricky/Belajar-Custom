Tipedata String

Pada Java Script tipe data String Bisa di Tulis dengan Cara
" Text " Menggunakan Petik 2, Atau
' Text ' Menggunakan Petik 1

Tapi Terdapat Masalah Jika Textnya Seperti ini :
" Hari Jum'at Akan di Lakukan Gladi Bersih Oleh Warga desa "Tanjung Tani" "

Maka Harus di akali dengan cara menggunakan Enscape atau / (Slash) Menjadi seperti ini
" Hari Jum'at Akan di Lakukan Gladi Bersih Oleh Warga desa \"Tanjung Tani\" "

terdapat Banyak Enscape Karakter Untuk JavaScript Seperti di Bawah ini :

Enscape Hasil
\ 0 Karakter NULL
\ ' Memberikan tanda string
\ " Memberikan tanda string
\\ Memberikan tanda string
\n Enter Kebawah
\t Untuk Tab
\b Backspace
.length Mengetahui Jumlah Karakter dari String
